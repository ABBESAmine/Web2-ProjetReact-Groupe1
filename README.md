# Etapes de mise en place du site

## Clonez le dépôt :
```bash
git clone https://github.com/ABBESAmine/Web2-ProjetReact-Groupe1.git
cd Web2-ProjetReact-Groupe1
```
## Installez les lib :
```bash
npm install
```
## Lancer l'appli :
```bash
npm run dev
```
